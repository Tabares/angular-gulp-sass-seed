#Angular Seed

## Instructions
------------------

1. Run `npm install`
2. Run `bower install`
3. Run `gulp` to start the file watcher
4. Open dist/index.html to view your page

## IMPORTANT

For the gulp to load angular files, components **MUST** be named with syntax like the following:
```
*.controller.js
*.directive.js
*.factory.js
*.module.js
```
